## گام اول – افزودن دو قابلیت جدید بدون رعایت اصول OOD

در این گام، دو قابلیت جدید به سیستم افزوده شد:
- سپهر نوروزی چاکلی: افزودن روش ارسال پیامک (SMS)
- محمدمهدی حسنی: افزودن روش پرداخت حضوری (On‑site Payment)

| ردیف | تغییرات مربوط به افزودن روش جدید ارسال پیام | | تغییرات مربوط به افزودن روش جدید پرداخت | |
|:---:|:-----------------------------:|:------------------------------:|:-----------------------------:|:------------------------------:|
|  | **کلاس تغییر یافته** | **توضیح کوتاه در مورد تغییر** | **کلاس تغییر یافته** | **توضیح کوتاه در مورد تغییر** |
| ۱ | MessageSender | افزودن یک تابع جدید برای ارسال پیامک به نام `sendSmsMessage` |  |  |

| ۳ | ReservationService | تغییر در متد ارسال اعلان رزرو برای اضافه‌کردن منطق ارسال پیامک |  |  |
